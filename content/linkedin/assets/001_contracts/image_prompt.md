# Image Generation Prompt

Create a clean, modern technical architecture visual for a LinkedIn post.

Subject:
A contract-first wearable data ingestion system.

Scene:
Show a simple left-to-right pipeline:

1. iOS Collector App
2. Sensor Adapters
   - Polar Verity Sense
   - Muse Athena
3. Stable Transport Contract
   - Session
   - Stream
   - Upload Chunk
4. Raw Storage
5. Parser / Normalizer
6. Visualization / Future ML

Design style:
- professional engineering diagram
- dark or neutral background
- subtle accent colors, not overly bright
- clean thin lines and rounded boxes
- minimal text
- readable on LinkedIn feed
- 16:9 aspect ratio
- no logos
- no brand names as logos
- no people
- no photorealistic devices required

Important visual idea:
Make the "Stable Transport Contract" box look like the central boundary of the system.
Show different sensor payloads flowing into the same stable contract, then raw storage before parsing.

Text to include:
- "Collector"
- "Adapters"
- "Transport Contract"
- "Raw Storage"
- "Parser"
- "Visualization / ML"

Avoid:
- Kubernetes imagery
- cloud provider logos
- overly complex network diagrams
- tiny unreadable text
- futuristic sci-fi style
- medical claims or hospital visuals

Overall mood:
Calm, practical, production-like engineering.
