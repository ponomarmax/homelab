# Image Generation Prompt

Create a clean, modern technical visual for a LinkedIn post.

Subject:
An iOS wearable collector skeleton that runs in simulator with a mock heart-rate stream.

Scene:
Show a simple left-to-right product-and-architecture composition:

1. iPhone simulator screen with a minimal app UI
   - title: Wearable HR Collector
   - state card
   - latest HR value changing
   - sample count
   - Start and Stop buttons
2. Collector Core
3. Device Adapter
4. Mock HR Stream Provider
5. Future Transport Boundary

Design style:
- professional engineering visual
- clean modern product style
- light or neutral background
- subtle blue, teal, and graphite accents
- thin connection lines
- rounded cards
- minimal readable labels
- 16:9 aspect ratio
- optimized for LinkedIn feed
- no logos
- no photorealistic Polar device
- no futuristic sci-fi look

Important visual idea:
Make the simulator UI feel real and grounded, but keep the architecture layers visible behind or beside it.
The mock stream should feel intentional, like a deliberate testing layer before real BLE integration.

Text to include:
- "iOS Collector"
- "Collector Core"
- "Device Adapter"
- "Mock HR Stream"
- "Future Transport"
- "Simulator-First"

Avoid:
- Bluetooth icons as the main focus
- backend servers
- cloud imagery
- medical claims
- tiny text
- cluttered wireframe aesthetics

Overall mood:
Calm, practical, production-like mobile engineering.
